BML Payment Gateway for WooCommerce

BML Payment Gateway for WooCommerce is a third-party WooCommerce plugin for receiving payments through the Bank of Maldives (BML) Payment Gateway.

Requirements

  • WordPress running on the host machine
  • WooCommerce plugin installed in WordPress
  • A valid merchant details provided to you by the Bank of Maldives

Installation

  1. Download the “bml-payment-gateway” plugin
  2. Go to plugins -> Add New in the sidebar of the WordPress Admin Page
  3. Upload the plugin zip file and Activate it
  4. Go to WooCommerce settings in the WordPress Admin Page
  5. Go to the 'Payments' tab that appears in the horizontal bar
  6. Turn on the "BML Payment Gateway" payment option and click on "Manage"
  7. Enter the Merchant ID, Acquirer ID, and Transaction Password provided by the BML. Customize other settings if you want
  8. BML Payment Gateway method should now show on the checkout page of your store
